WebServer

1) A web server is a computer system that processes requests via HTTP
2) The primary function of a web server is to store, process and deliver web pages to clients.

Web Client

1) A web browser is a software application for retrieving, presenting and traversing information resources on the World Wide Web.
2) The primary purpose of a web browser is to bring information resources to the user ("retrieval" or "fetching"), allowing them to view the information ("display", "rendering"), and then access other information ("navigation", "following links").
3) Web Client and Web Browser are the same
